# Single Deep CFR for Texas Hold’em

本项目实现了一个基于单深度反事实遗憾最小化算法的德州扑克 AI，参照Eric Steinberger的论文 _Single Deep Counterfactual Regret Minimization_，增加了动作空间。

[![arXiv](https://img.shields.io/badge/arXiv-1901.07621-b31b1b.svg)](https://arxiv.org/abs/1901.07621)

在工程实现上，SD-CFR 吸收了深度强化学习领域的成功经验，以解决传统博弈论算法在庞大状态空间下的维度爆炸问题：

- **Monte Carlo(MC-CFR)**：传统 CFR 需要像穷举法一样遍历整棵博弈树 。本项目采用外部采样（Traverser探索所有合法动作，其他按照概率采样）抽取轨迹，降低单次计算的复杂度 。

- **深度学习**：抛弃了传统 CFR 庞大的查表法，引入了深度神经网络（价值网络）来近似拟合优势值（也就是归一化的遗憾值regret）。

- **经验回放池**：将采样得到的瞬时遗憾值数据存入一个内存缓冲区 $B^v$，并使用蓄水池采样 (Reservoir Sampling) 机制进行动态更新 。这打破了数据的时间相关性，使神经网络能进行稳定的 Batch 梯度下降。

**CFR 与 RL 的差异**

1. **优化目标：最大化收益 vs 逼近纳什均衡。** RL 致力于在马尔可夫决策过程中找到最优策略以最大化期望回报；而 CFR 面对的是不完美信息零和博弈，其目标是最小化可剥削度，逼近纳什均衡，即确保在面对最完美的对手时也能立于不败之地 。

2. **拟合指标：Q-Value vs 反事实遗憾值。** RL 网络评估的是动作的未来期望得分。CFR 的价值网络学习的是“反事实遗憾值 (Counterfactual Regrets)”——即假设对手策略不变，如果我当时绝对选择了另一个动作，我能多赚多少筹码。

3. **最终策略：最新网络 vs 历史平均。** 在 RL 中，训练结束时的最新网络往往是最强的；但在 CFR 中，单次迭代的策略 $\sigma^t$ 会在博弈树中不断震荡跳跃，真正能收敛到纳什均衡的是所有历史迭代策略的加权平均值*。

## SD-CFR 核心技术与原理

传统的 Deep CFR 算法 使用了两个独立的神经网络：一个近似遗憾值，另一个平均策略网络专门用来逼近历史策略的平均值 。

Eric Steinberger 提出的 SD-CFR 架构：

- **摒弃平均策略网络**：SD-CFR 直接去掉了用来学习平均策略的第二个神经网络 。这不仅节省了大量的计算开销，更从根本上消除了由于策略缓冲区容量有限而引入的采样误差和近似误差 。

- **历史网络池与轨迹采样**：为了在不使用平均网络的情况下输出“平均策略”，SD-CFR 将每轮迭代训练好的价值网络保存到一个历史网络池 $B^M$ 中 。当对局从根节点开始时，AI 会从池中抽取一个网络 $\hat{D}^t$，其被抽中的概率与该网络的迭代轮次 $t$ 成正比（线性加权） 。

- **理论证明**：在整局游戏中，AI 将始终使用这一个被抽中的网络进行决策 。**轨迹采样**机制，不仅天然满足了 Linear CFR 的加权约束，还在数学上被严格证明：只要价值网络拟合准确，SD-CFR 就能**毫无误差**地代表真实的平均策略 。

## 训练结果 (基于 100 轮迭代的早期原型)

原论文的动作空间是{fold, call, raise}，本项目增加为{fold, call, raise 0.2 pot, raise 0.5 pot, raise 1 pot}。下注规则是：底注为10，一轮最多加注2次。在使用较小参数（100 轮迭代，每轮 200 次采样）对模型进行初步训练后，AI 虽然仍处于非常早期的版本，但已经展现出一些博弈逻辑。在使用训练好的AI自博弈后，选出了几个有意思的案例。

#### 案例一：空气牌反诈唬

- **底牌**：P0 `Tc 2c` (烂牌) vs P1 `Qc 9s` (中等边缘牌)

- **过程**：
  
  玩家 P0 动作分布: `[0.217 0.356 0.29  0.137 0.   ]`  ，选择了: 过牌
  
  P1 敏锐地嗅到了 P0 翻前的软弱过牌，动作分布直接给出：`[0., 0.321, 0., 0., 0.679]`。它有近 68% 的概率选择满池加注，试图用巨大的下注体量直接买下底池。
  
  面对满池的巨大压力，手里同样只有空气牌 `T2` 的 P0，其动作分布居然是 `[0.068, 0.021, 0.323, 0.3, 0.288]`。弃牌率仅 6.8%，反而打出了高概率的反加注 (Re-raise)！P1 面对反击，瞬间 100% 弃牌逃跑。

- **逻辑**：这是早期 CFR 左手打右手产生的策略制衡。P0 的网络知道 P1 很大概率在诈唬，因此学会了“即使自己是烂牌，勇敢反加注，也有机会强迫诈唬的对手弃牌”。

#### 案例二：深层博弈下的极致压榨 (The River Value Jam)

- **底牌**：P0 `7c Ac` vs P1 `Qc 3d`

- **过程**：
  
  双方拿着 A 高牌和 Q 高牌一路火拼，底池被炒到了极其夸张的 27216，体现了早期 AI 在大底池评估时的混乱。
  
  **高光时刻**：当河牌发出 `Ad`，P0 击中顶对。在上一条街原本分布依然散乱的 P0，面对这张 A，其策略瞬间极化，动作分布锁定在 **`[0., 0., 0., 0., 1.]`**（100% 满池加注）。

- **逻辑**：当神经网络感知到自己拥有绝对的牌力优势时，没有任何过牌引诱的打算，直接执行最大化的价值提取。

#### 案例三：翻后施压

- **底牌**：P0 `Tc Ac` (A 高牌) vs P1 `8h Ks` (K 高牌)

- **过程**：
  
  翻前P0先行动，位置弱势选择过牌埋伏，P1给出`[0.    0.504 0.35  0.146 0.   ]` 选择打出小注试探后，p0牌力优势面对试探直接check raise，P1选择跟注。
  
  翻牌面对 `3d 2s 2c` 极其干燥的翻牌面，P0 的策略只有`[0.587 0.    0.413 0.    0.   ]`试探性地打出小注 (`Raise 0.2P`)。
  
  P1 只有 K 高牌，但网络给出了 `[0.144, 0.435, 0.233, 0.187, 0.]`，果断进行半池反加注。
  
  P0 面对反击，动作变为 **`[1., 0., 0., 0., 0.]`**，100% 果断弃牌。

- **AI 逻辑**：P1 学会了利用干燥牌面结构对翻前强势的p0反击，而 P0 学会了止损：**“没中牌，面对重注就跑”**，绝不用 A 高牌支付未知的反击。
